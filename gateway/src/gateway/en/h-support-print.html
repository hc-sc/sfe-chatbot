Support for printers

This service covers support for both network printers and local printers .

Network printer

A network printer is a device on the HC / PHAC network that can support the printing needs of multiple users at the same time.

Local printer

A local printer directly connects to a specific computer and it is only that computer that can access or be serviced by the printer.

Network printers, which are often multi-function devices (i.e. perform more than one function; print scan, copy), are high-usage equipment. When issues are experienced that interrupt the printing ability of even one of these machines, large numbers of users can be impacted. Be sure to report any printer device malfunctions as soon as possible. While waiting for the device to be fixed, you can use the iPrint Client to get access to a different network printer.
