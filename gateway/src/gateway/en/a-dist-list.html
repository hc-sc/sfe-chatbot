Outlook Email distribution lists

Outlook email distribution lists serve as means to communicate regularly with large groups of users. The list is a collection of email addresses that allow you to communicate with multiple people in one email message. A message sent using a distribution list goes to all recipients on the list.

Email distribution lists appear in the Global Address List in your Outlook Address Book and can be used by all users, unless access has been restricted.

Use email distribution list if:

  • Many persons will need to use the list
  • The number of members in the group is large (over 120 members)
  • The list members are all in the Government of Canada (GC) Outlook system.

Create or modify distribution list

Use this service to:

  • create new distribution list
  • rename existing distribution list
  • delete existing distribution list
  • modify distribution list owner/manager

Note that you need to be the "Owner/Manager" of the new or existing distribution list for this request to be completed. If you are not the owner of the list, but need an existing list to be renamed or deleted, contact the owner of the list directly.

Change the list membership

If you are the owner of the list, use Outlook to change the group membership. For instructions on how to add and remove members (email addresses), consult How to guide – Outlook rules, notifications, contact groups and distribution lists. If you are not the owner of the list, contact the owner of the list directly.

Access departmental email distribution lists

Departmental email distribution lists are distribution lists used for large scale departmental communications (e.g. All HC / PHAC employees, CFOB employees). Use and access of these lists is limited and restricted to departmental and functional heads and their delegates. Access to a departmental distribution list must be granted by the list owner.

Note

  • You must follow GC email standards and guidelines when naming departmental email distribution list.
