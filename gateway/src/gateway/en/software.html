Software / applications

Software includes everything that runs on your computer / tablet; from the operating system (e.g Windows), necessary for the functioning of your computer to commercially available productivity software (available based on business line requirements and departmental standards e.g. Microsoft Office, Adobe Acrobat) as well as internally developed or commercially adapted business solutions (e.g. PeopleSoft, GCdocs).

Software services include getting and installing new software / applications, technical support, and return / remove or transfer of installed software / applications.
