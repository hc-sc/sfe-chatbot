Get software

A large number of software / applications have been tested and approved for use on the network to support HC / PHAC business needs.

Use the "Search software" on this page to determine if the software / application you need has been pre-approved. If your software is found, follow the "Action" link to get the software / application.

If the software you are searching for is not found, it means that it has not been previously approved for use on the HC / PHAC network. You must follow the Not pre-approved software process to request this software. This includes upgrading existing software to a new version.
